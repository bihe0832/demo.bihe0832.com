JSbridge测试Demo

该Demo主要用户子勰的Jsbridge测试，测试步骤如下：

  • 使用内置浏览器打开URL：https://cdn.bihe0832.com/tools/jsbridge/index.html
  • 打开浏览器的开发者模式，在console中输入测试命令，例如： call('getAppInfo', {packagename:"com.tencent.mm"}, (res) => { console.log(res) })
  • 在console中查看返回结果
  • 想要了解更多信息，可以点击链接 https://android.bihe0832.com/doc/tools/android_jsbridge.html
